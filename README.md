# Scoop Bucket para Omusic

Este es el bucket oficial de [Scoop](https://scoop.sh) para [omusic](https://github.com/AlexCas/omtube).

## Uso

Para instalar `omusic` en Windows utilizando Scoop, abre tu terminal y ejecuta:

```powershell
scoop bucket add scoop-omusic https://github.com/AlexCas/scoop-omusic.git
scoop install omusic
```

> **Nota:** Este repositorio se actualiza de manera automática mediante GitHub Actions cada vez que hay un nuevo Release en el repositorio principal.
